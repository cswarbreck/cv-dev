#4.1Portfolio

Chess with Tony

Tony Niccoli is a chess teacher in London, running a small business. He needed a site that could be a focal point for delivering information, handle inquiries and would be easy to update. He didn't want to use Wordpress or an equivalent, so I built a site around KeystoneJS, an open source CMS using NodeJS with no on-going costs or tech support needed.
